Stress Testing Framework — 6 Scenarios
Each scenario simulates a different adverse market / infrastructure condition · all 6 must pass for CERTIFIED
SCN-01
Flash Crash
SIMULATION
XAUUSD drops 8% in 90 seconds, then recovers 60% over 4 minutes
HISTORICAL BASIS
Aug 2020 silver flash, Mar 2020 COVID crash
TESTS
Kill-switch, stop-loss fills, position flatten
RECOVERY
Halt trading 5 min · resume with reduced size
PASS MDD
≤ 12% (vs 5% live target — 2.4× headroom)
SCN-02
High Spread
SIMULATION
Spread widens to 5× baseline for 30 min (simulated NFP event)
HISTORICAL BASIS
NFP/FOMC/CPI release ±2 min windows
TESTS
News filter, order rejection, cost cap
RECOVERY
Auto-suppress entries · resume after spread < 2× baseline
PASS COST DRAG
≤ 45% (vs 35% normal — 10 pp headroom)
SCN-03
Server Lag
SIMULATION
Tick-to-execution latency 2× budget for 60s (300 ms P99)
HISTORICAL BASIS
VPS CPU contention, network congestion
TESTS
Stale-signal veto, backpressure, drop-tick logic
RECOVERY
Risk engine auto-vetoes stale signals · alert ops
PASS STALE %
≥ 95% signals correctly vetoed (no stale fills)
SCN-04
Broker Disconnect
SIMULATION
MT5 socket drops for 90s, then auto-reconnects
HISTORICAL BASIS
Broker-side maintenance, ISP outage
TESTS
Reconnect logic, position sync, missing-tick handling
RECOVERY
Auto-reconnect < 5s · position reconciliation · flat check
PASS RECONCILE
100% positions reconciled, 0 phantom orders
SCN-05
Extreme Volatility
SIMULATION
ATR spikes to 4× 30-day avg for 2 hours
HISTORICAL BASIS
Geopolitical event, central bank surprise
TESTS
Vol regime detection, position sizing, exposure cap
RECOVERY
Auto-reduce size 50% · widen stops · halt new entries
PASS MDD
≤ 10% during vol spike (vs 5% live — 2× headroom)
SCN-06
Gap Open
SIMULATION
Sunday open gaps +3% / −3% from Friday close
HISTORICAL BASIS
Weekend event risk (geopolitical, news)
TESTS
Weekend position policy, gap-risk buffer, stop fills
RECOVERY
Flat by Friday 22:00 GMT · no weekend exposure
PASS GAP LOSS
≤ 2% account equity (no weekend gap exposure)
