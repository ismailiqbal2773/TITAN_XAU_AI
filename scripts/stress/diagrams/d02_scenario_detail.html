SCN-01 · FLASH CRASH DETAIL
8% drop in 90 seconds
MAGNITUDE
−8% price drop (e.g., 2000 → 1840 USD/oz)
DURATION
90 s drop · 4 min partial recovery (+5%)
INJECTION
Random time during London session (08:00-17:00 UTC)
EXPECTED BEHAVIOR
Kill-switch triggers on MDD ≥ 5% · all positions flattened in <500 ms
RECOVERY
Halt 5 min · resume with 50% size · 24h monitoring
FAILURE
MDD > 12% OR kill-switch > 500 ms OR phantom positions
SCN-02 · HIGH SPREAD DETAIL
5× baseline spread for 30 min
MAGNITUDE
Spread = 5 × baseline P50 (e.g., 0.18 → 0.90 USD)
DURATION
30 min (simulating extended NFP reaction window)
INJECTION
Spread overlay applied to tick stream, prices preserved
EXPECTED BEHAVIOR
News filter suppresses new entries · existing positions held with widened stops
RECOVERY
Resume new entries when spread < 2× baseline for 60s
FAILURE
Cost drag > 45% OR new entry during widened spread OR stop-out
SCN-03 · SERVER LAG DETAIL
300ms P99 latency for 60s
MAGNITUDE
Tick-to-exec latency = 300 ms P99 (2× the 150 ms budget)
DURATION
60 s (typical CPU contention / network blip)
INJECTION
Network simulation layer adds 150 ms delay to broker round-trip
EXPECTED BEHAVIOR
Risk engine auto-vetoes "stale" signals (timestamp > 150 ms old)
RECOVERY
Resume normal execution when latency < 200 ms for 30s
FAILURE
Stale-veto rate < 95% OR any stale fill OR order queue overflow
