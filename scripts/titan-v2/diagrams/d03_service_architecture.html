EVENT BUS · NATS JETSTREAM
All services communicate via async events · zero shared state · backpressure-aware
Topics: ticks, signals, orders, fills, risk_alerts, regime_change, license_events, audit. Each topic has 3-day retention with replay. Subscribers can replay from any offset. Backpressure handled via JetStream consumer ack + max-deliver limits.
