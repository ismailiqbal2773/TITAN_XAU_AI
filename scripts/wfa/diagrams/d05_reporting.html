ARCHIVE
All reports to s3://titan-wfa/{strategy}/{version}/{timestamp}/ · 7-year retention · RSA-2048 signed manifest
DISPATCH

VERSIONING

COMPARISON
Each WFA compared against last 5 WFA runs of same strategy. > 10% WFE drop = regression P1 alert.
CADENCE
Quarterly scheduled WFA on every live strategy · On-demand via CLI / REST · Pre-deployment mandatory

REPORT CONTENTS — TECHNICAL TIER (FULL DETAIL)
SECTION 1
WFA configuration — methodology (anchored/rolling), train/val/test window lengths, fold count, dataset version, cost profile version.
SECTION 2
Per-fold results table — IS Sharpe, OOS Sharpe, OOS MDD, OOS return, parameter set selected, cost breakdown for each fold.
SECTION 3
Parameter evolution — selected parameter values per fold + stability analysis (max/min/median/stdev). Detects parameter-chasing.
SECTION 4
Aggregate OOS metrics — WFE, aggregate OOS Sharpe / MDD / CAGR, fold consistency, cost drag, all compared to IS baselines.
SECTION 5
Equity curves — IS, OOS, and IS+OOS combined. Visual overfitting detection (large IS-OOS gap = overfit).
SECTION 6
Regime attribution — per-fold regime distribution (trend/range/volatile/news) and per-regime OOS performance.
SECTION 7
Certification verdict — 3-band (CERTIFIED/CONDITIONAL/REJECTED), aggregate score, hard veto triggers fired (if any), waiver IDs (if any).
