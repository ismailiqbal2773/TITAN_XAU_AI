DEPLOYMENT DOCUMENTATION — SUMMARY
Systemd service + PostgreSQL + Grafana + Prometheus
Step 1: Provision 4-vCPU VPS (8GB RAM, 100GB SSD) with Ubuntu 22.04 LTS. Step 2: Install PostgreSQL 15 + TimescaleDB extension. Step 3: Create database `titan_ceo`, run schema DDL (10 tables + hypertables + audit triggers). Step 4: Install Python 3.12 + dependencies (asyncpg, numpy, nats-py, prometheus-client). Step 5: Clone CEO repo, configure /etc/titan/ceo.yaml (NATS URL, DB DSN, alert webhooks). Step 6: Install systemd service file (/etc/systemd/system/titan-ceo.service), enable + start. Step 7: Import Grafana dashboard JSON (8 panels: 6 health scores + system status + action log). Step 8: Configure Prometheus scrape (ceo:9100/metrics). Step 9: Configure PagerDuty integration (P1 RED, P2 YELLOW). Step 10: Run smoke test (feed 100 synthetic trades, verify GREEN status + DB rows).
