Rolling Windows — 4 Sizes × 8 Metrics × Per Model
Multi-scale view · short-term sensitivity (50) + long-term stability (500) · bounded O(N) memory
4 ROLLING WINDOWS
Per model per metric
W50
short-term
50-Trade Window
Scope: last 50 trades (~1 week). Use: detect acute degradation, sudden regime shifts, immediate broker issues.
decay: none · memory: 50×8×4 models = 1600 floats · refresh: every trade
W100
medium-term
100-Trade Window
Scope: last 100 trades (~2 weeks). Use: confirm W50 signals, smooth short-term noise, detect emerging drift.
decay: none · memory: 100×8×4 models = 3200 floats · refresh: every trade
W250
long-term
250-Trade Window
Scope: last 250 trades (~1 month). Use: stable baseline, overfitting detection (vs IS baseline), drift confirmation.
decay: none · memory: 250×8×4 models = 8000 floats · refresh: every trade
W500
strategic
500-Trade Window
Scope: last 500 trades (~2 months). Use: strategic baseline, persistent drift, structural changes, regime transition.
decay: none · memory: 500×8×4 models = 16000 floats · refresh: every trade
