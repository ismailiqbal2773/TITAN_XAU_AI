Monte Carlo Framework — Methodology Overview
6-stage pipeline · 10,000 simulations · 3 randomization dimensions · Survival Score certification
STAGE 1
Load Backtest Results
Import per-trade ledger from Backtest Framework (M16). Each trade: entry, exit, P&L, regime, costs.
Input: trade ledger CSV
Trades: 200-800 typical
→
STAGE 2
Define Randomization
Configure 3 randomization dimensions: trade order, slippage, spread. Set distributions and bounds.
Dimensions: 3
Seed: reproducible
→
STAGE 3
Run 10,000 Simulations
For each sim: shuffle trades, sample slippage, sample spread. Compute equity curve + metrics.
Sims: 10,000
Runtime: ~6 min
→
STAGE 4
Aggregate Distribution
For each metric (Sharpe, MDD, CAGR, RoR), compute P5/P25/P50/P75/P95 across 10k sims.
Metrics: 8
Percentiles: 5
→
STAGE 5
Survival Score
Compute Survival Score: fraction of sims that remained profitable + met risk constraints.
Score: 0-100
Target: ≥ 95
→
STAGE 6
Certification
Apply pass/fail criteria. Produce 3-band verdict: CERTIFIED / CONDITIONAL / REJECTED.
Verdict: 3-band
Report: 3-tier
CORE FORMULA
SurvivalScore = (Sims_Profitable AND MDD ≤ 8% AND Sharpe ≥ 1.0) / Total_Sims × 100
Simulation_i: equity_curve( shuffle(trades), sample_slippage(trades), sample_spread(trades) )
SIMULATIONS
10,000
Per strategy evaluation · seed-reproducible
RANDOM DIMENSIONS
3
Trade order + Slippage + Spread
SURVIVAL TARGET
≥ 95%
Fraction of sims profitable AND within risk
RUNTIME
