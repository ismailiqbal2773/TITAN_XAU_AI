Institutional Backtesting Framework — Architecture
5 cost components · 4-layer data pipeline · 6-stage testing process · 24 metrics · 3-band certification
Cost Components (Realistic Edge)
5 LAYERS
C1 · TICK DATA
Bid/Ask tick stream (not OHLC)
Resolution: 100 ms granularity, sourced from broker true tick history. Captures intra-bar reversals invisible to 1-min candles.
C2 · VARIABLE SPREAD
Time-varying spread model
Real broker spread per tick. News widening ×3-10×. No fixed-spread assumption (the #1 backtest lie).
C3 · COMMISSION
Round-turn $/lot
Per-broker commission structure: $3.50–$7.00 RT/lot for ECN, 0 for market-maker (already in spread). Verified per fill.
C4 · SWAP
Overnight financing per position
Long XAUUSD: −2.5 to −6.0% annualized. Short: +0.5 to −1.5%. Triple swap Wednesdays. Compounded daily.
C5 · SLIPPAGE
Realistic fill model
P50/P90/P99 slippage distribution per broker, calibrated from live fills. Market impact for size > 5 lots.
Pipeline Layers
4 LAYERS
L1 · DATA INGESTION
Tick + Spread + Calendar
Load broker tick history (Dukascopy, broker export), economic calendar (NFP/FOMC/CPI), swap schedule from broker spec sheet.
L2 · COST ENGINE
Per-tick P&L attribution
Apply spread, commission, swap, slippage to each simulated fill. Produces realistic equity curve vs idealized price-only curve.
L3 · EXECUTION SIMULATOR
Order matching engine
Simulate MT5 order routing: market, limit, stop, OCO. Partial fills, requotes on news, latency 100-200 ms, margin calls.
L4 · METRICS & REPORTING
24 metrics + 3 reports
Sharpe/Sortino/Calmar, MDD, profit factor, win rate, expectancy, cost drag, regime breakdown. Output: executive / technical / regulatory.
