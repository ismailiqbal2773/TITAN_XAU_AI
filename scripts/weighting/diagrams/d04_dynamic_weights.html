ADAPTATION — WEIGHTS CHANGE EVERY 60 SECONDS
No hardcoded regime → weight mapping. Weights EMERGE from performance.
The same 4 algorithms run every cycle (60s). The Meta-Bandit selects which algorithm to use based on recent weight-quality in the current regime. If Transformer starts underperforming in trend regime, its weight drops within 60s — no human intervention, no code change. If a new regime pattern emerges that no algorithm handles well, the Meta-Bandit explores all 4 algorithms (Thompson Sampling's exploration) until it finds the best one. This is true adaptive intelligence — the system learns which model to trust, in which regime, using which algorithm, all in real-time.
