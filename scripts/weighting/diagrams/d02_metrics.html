7 Performance Metrics — Per Model
7 metrics computed per model per cycle · NumPy vectorized · feeds into 4 weighting algorithms
M1 · ACCURACY
Model Accuracy
Directional accuracy: did model predict correct trade direction?
accuracy = correct_directions / total_predictions
Window: W100 (100 trades)
M2 · PROFIT FACTOR
Profit Factor
Gross profit / gross loss for trades where this model was primary contributor.
PF = gross_profit / gross_loss
Window: W100
M3 · SHARPE
Sharpe Ratio
Risk-adjusted return of model's attributed trades.
Sharpe = mean(R) / std(R) × √252
Window: W250
M4 · DD CONTRIBUTION
Drawdown Contribution
How much this model contributed to system MDD. Lower = better.
DDC = model_attributed_loss / total_system_MDD
Window: W250
M5 · SLIPPAGE SENS
Slippage Sensitivity
Correlation between model's trade frequency and slippage cost.
SlipSens = corr(trade_freq, slippage_cost)
Window: W100
M6 · LATENCY SENS
Latency Sensitivity
How much model's Sharpe degrades when latency increases.
LatSens = Sharpe_low_latency - Sharpe_high_latency
Window: W250
M7 · REGIME PERF
Regime Performance
Per-regime win rate for this model. Drives regime-conditional weighting.
RegimePerf = WR(regime=current) over W100
Window: W100 per regime
